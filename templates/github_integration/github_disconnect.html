    GitHub Hesap Bağlantısını Kaldır

    Dikkat! GitHub hesap bağlantınızı kaldırmak üzeresiniz.

    Bu işlem, GitHub ile senkronize edilmiş projeleri ve görevleri doğrudan etkilemeyecektir. Ancak bundan sonra GitHub ve uygulama arasında senkronizasyon işlemleri yapılamayacaktır.

    Lütfen şu konularda dikkat edin:
    • Bağlı projelere ait GitHub repository referansları korunacaktır
    • Görevlerin GitHub issue bağlantıları korunacaktır
    • Ancak artık projeler ve görevler GitHub ile senkronize edilemeyecektir
    • İstediğiniz zaman GitHub hesabınızı yeniden bağlayabilirsiniz
